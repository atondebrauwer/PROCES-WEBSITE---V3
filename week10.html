WEEK 10

Doel van de week

Deze week heb ik CSS toegepast op mijn website. Nadat de basisstructuur van de HTML-pagina’s klaar was, ben ik begonnen met het toevoegen van styling. Ik heb gewerkt aan kleuren, typografie en eenvoudige layout, zodat de website visueel beter aansloot bij mijn ontwerp.

resultaat

Reflectie

Het toepassen van CSS zorgde ervoor dat mijn website echt tot leven begon te komen. Soms was het zoeken naar de juiste instellingen, vooral bij typografie en uitlijning, maar door te experimenteren kreeg ik steeds meer inzicht in hoe CSS werkt. Deze week hielp me om beter te begrijpen hoe ontwerp en code samenkomen en gaf me meer vertrouwen om mijn website verder te verfijnen.
