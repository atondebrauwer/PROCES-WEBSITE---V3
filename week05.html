WEEK 5

Doel van de week

Het doel van deze week was om de basis van HTML en CSS te leren en deze toe te passen op mijn eigen ontwerp. Ik wilde begrijpen hoe een webpagina is opgebouwd en hoe ik mijn ontwerp uit Figma kan omzetten naar een eenvoudige, werkende HTML-pagina.

resultaat

Reflectie

Het bouwen van mijn eerste HTML-pagina was uitdagend maar ook motiverend. Sommige onderdelen, zoals de juiste structuur en het correct koppelen van CSS, vroegen extra oefening. Door te experimenteren werd het steeds duidelijker hoe kleine aanpassingen in code een groot effect kunnen hebben op het ontwerp.
